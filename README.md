# itchat-demo

基于 chatgpt-on-wechat 的itchat 写的demo，只实现了基本的登陆、自动回复、api推送，登陆、登出调用api, 基于微信网页版协仪-uos头

# Install

```bash
# 创建局部 python3 依赖环境
python3 -m venv ./venv

# 切换至当前依赖环境
source ./venv/bin/activate

# 安装依赖
python3 -m pip install -r requirements.txt

# 启动程序
python3 main.py
```
